Введение

Markdown — это простой язык разметки для быстрого форматирования текста (заголовки, списки, жирный/курсив), который легко преобразуется в HTML и используется для создания документации, постов в блогах и веб-контента, позволяя сосредоточиться на тексте, а не на сложном коде.

---

Основные возможности Markdown

Markdown позволяет создавать заголовки разных уровней, маркированные и нумерованные списки, выделять текст жирным или курсивом, вставлять ссылки и изображения, оформлять цитаты и блоки кода. При этом синтаксис остаётся интуитивно понятным и хорошо читается даже без преобразования в форматированный вид.

***

Пример форматирования

# Заголовок 1
## Заголовок 2
### Заголовок 3
